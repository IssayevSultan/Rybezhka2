Ваша задача это создать два блока, расположить их друг возле друга и в эти два блока поместить несколько input и добавить соответственно стили к каждому из input. Чтобы расположить два блока друг напротив друга сделайте два div'a и добавьте к ним стили: float, width и можно дополнительные стили уже просто для красоты. Теперь в каждый div поместите различные input с атрибутом type (разные значения). Теперь выберите input в css при помощи id или класса или же по имени (input['name']). Добавьте отступы (padding), backgroundcolor, color, border-radius и любые другие свойства, которые смогут украсить ваш input
